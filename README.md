# 🚀 CAMP2026 计划

> “一个旨在融合三大知识领域的精英训练蓝图：<br>
> 工程师的**系统之力 (CS)**, 架构师的**算法之智 (AI)**, 理论家的**数学之核 (Math)**。”

---

## 1. 计划宣言 (The Manifesto)

**CAMP2026** 是一个高强度的个人学习蓝图，旨在系统性地构建一个横跨计算机科学 (C)、人工智能 (A) 和数学 (M) 的精英知识体系。

* **核心定位**: 成为 “CS-AI-Math” 复合型高精尖工程人才。
* **目标岗位**: 顶尖 AI 领域 (AI-Infra, 算法岗) 或量化金融领域 (Quant Developer, 算法岗)。
* **核心战略**: 以“**开发技术为基本路线与核心**”，构建坚实的工程能力（高下限）；以“**论文产出为转向算法岗的契机**”（高上限）。

## 2. 蓝图核心 (The Blueprint)

计划分为三个核心板块，每个板块包含约 10 门“金牌课程”或同等深度的模块。

---

### 🌌 III. 数学 (Math) - 理论护城河

此板块是“核心护城河”，为 AI 和 Quant 提供理论深度和创新源泉。

* **1. 分析学 (基石)**
    * 分析学功底 (实分析, 泛函分析, 测度论)
* **2. 概统 (随机)**
    * 应用随机分析 (随机过程, **SDEs**, 伊藤引理)
    * 应用统计 (统计推断, 假设检验)
* **3. 计算 (数值)**
    * 矩阵计算与分析 (SVD, 特征值)
    * 现代数值分析 (逼近, 插值)
    * 方程数值解 (PDE/ODE 数值解, FEM/FDM)
* **4. 优化与控制 (算法核心)**
    * 离散优化 (组合优化)
    * 流形优化 (AI 几何)
    * 最优传输 (OT, 用于 GenAI)
    * (随机)最优控制 (HJB 方程, 用于 RL/Quant)
